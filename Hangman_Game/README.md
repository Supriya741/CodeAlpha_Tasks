# 🎮 Hangman Game using Python

This project is an interactive Hangman Game developed using Python as part of the CodeAlpha Python Programming Internship.

## ✨ Features
- Random word selection
- Remaining attempts tracker
- Guessed letters display
- Win/Lose conditions
- Interactive emoji-based UI

## 🛠 Technologies Used
- Python

## 📚 Concepts Used
- Random Module
- Lists
- Loops
- Conditional Statements
- Strings

## ▶ How to Run

```bash
python hangman_game.py
```

## 🎯 Game Rules
Guess one letter at a time
You have limited attempts
Correct guesses reveal letters
Wrong guesses reduce attempts

## 🎯 Outcome

This project helped in improving problem-solving skills and understanding Python game logic.
