- Guessed letters display
- Win/Lose conditions
- Interactive emoji-based UI

 🛠 Technologies Used
- Python

 📚 Concepts Used
- Random Module
- Lists
- Loops
- Conditional Statements
- Strings

 ▶ How to Run

```bash
