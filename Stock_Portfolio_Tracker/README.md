
# 📈 Stock Portfolio Tracker using Python

This project is a smart stock portfolio tracker developed using Python as part of the CodeAlpha Python Programming Internship.

- Portfolio summary generation

## 📚 Concepts Used

- Loops
- Conditional Statements
- Arithmetic Operations
- File Handling
