
# 📈 Stock Portfolio Tracker using Python

This project is a smart stock portfolio tracker developed using Python as part of the CodeAlpha Python Programming Internship.

## ✨ Features
- Tracks stock investments
- Calculates total investment value
- Portfolio summary generation
- File handling support
- User-friendly interface with emojis

## 🛠 Technologies Used
- Python

## 📚 Concepts Used
- Dictionary
- Loops
- Conditional Statements
- Arithmetic Operations
- File Handling

## ▶ How to Run

```bash
python stock_portfolio_tracker.py
```

## 📌 Available Stocks
AAPL
TSLA
GOOG
AMZN

## 📂 Output File

The project automatically saves portfolio details in:

portfolio.txt
## 🎯 Outcome

This project helped in understanding dictionaries, calculations, loops, and file handling in Python.
